# cs1534-as1
Assessment 1
Visit Brixton
This project is about creating an in-depth website for my home town and making sure that it has all of its links working and showing the ability to use html and css. Brixton is where I am from and so the website features the history and lifestyle of brixton in a purple, black and white format as those are my favourite colours.

You can run my program by simply downloading the files onto your computer in a singular folder and access it on any web browser via your pc. 
Navigating between pages is simple at the top of the website below the "Brixton Guide" indicator, you can toggle between the linked pages by clicking on the required buttons below. The page accessed is indicated by the colour of the button, whether it is purple or empty.

I found the most difficult part of the assignment to be getting my writing to successfully wrap with each other, especially trying to get it to be a 7:5 column ratio, I also struggled with the original colour scheme and layout of the website which is why I was using the Blackboard Aberdeen's css colours to begin with. I decided against making a more complex table style navigation system as I felt as if it was unecessary on such a small website, which is why i went with the button format.

Overall I tried my hardest with this website and to ensure its as functional as possible, I used Wikipedia, Google Gemini and the official sites listed in the webpage to create the big blocks of text for each of the sections as permissed.
